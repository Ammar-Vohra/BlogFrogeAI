# 📌 About The Project

**BlogForge AI** is an intelligent AI-powered blog automation engine that converts a simple topic into a complete professional blog article using advanced **multi-agent architecture**.

It automatically:

✅ Researches trending topics  
✅ Plans blog structure  
✅ Writes sections in parallel  
✅ Generates AI images  
✅ Adds citations  
✅ Exports final Markdown blogs

# 🧠 Multi-Agent Architecture

                ┌─────────────────┐
                │   User Topic    │
                └────────┬────────┘
                         ↓
                ┌─────────────────┐
                │ Router Agent    │
                │ Need Research?  │
                └────────┬────────┘
                         ↓
        ┌────────────────────────────────┐
        │ Research Agent (Tavily Search)│
        └────────────────┬──────────────┘
                         ↓
                ┌─────────────────┐
                │ Planner Agent   │
                │ Creates Outline │
                └────────┬────────┘
                         ↓
         ┌─────────────────────────────────┐
         │ Parallel Writer Worker Agents   │
         └────────────────┬────────────────┘
                          ↓
                ┌─────────────────┐
                │ Reducer Agent   │
                │ Merge Sections  │
                └────────┬────────┘
                         ↓
                ┌─────────────────┐
                │ Image Generator │
                └────────┬────────┘
                         ↓
                ┌─────────────────┐
                │ Final Blog File │
                └─────────────────┘

# ✨ Features
### 🤖 Smart Blog Generation

Generate multiple blog types:

* Explainer Blogs
* Tutorials
* News Roundups
* Product Comparisons
* System Design Articles

---

### 🔍 Live Research Support

Uses Tavily Search API for:

* Recent trends
* Fresh facts
* Real sources
* Better factual accuracy

---

### ⚡ Parallel AI Writing

Each blog section is generated independently using worker agents for faster performance.

---

### 🖼️ AI Image Generation

Automatically inserts custom AI-generated images relevant to the content.

---

### 📄 Export Ready

Blogs are exported as clean Markdown files ready for:

* GitHub Pages
* Medium
* Dev.to
* WordPress
* Notion
